Introduction¶

The data comes from the FBI's National Instant Criminal Background Check System. The NICS is used by to determine whether a prospective buyer is eligible to buy firearms or explosives. Gun shops call into this system to ensure that each customer does not have a criminal record or isnt otherwise ineligible to make a purchase. The data has been supplemented with state level data from census.gov.

Gun Data:

  1. Has the overall purchase of guns increased over the years?
  2. Which state has the highest growth in gun registrations?
  3. Which Gun type is preferred?
  4. Which Top 5 States have the most the gun purchase related activity?

Census Data:

  1. What census data is most associated with high gun per capita?
  2. What can be found about the top 5 census data on the states with maximum gun registrations?

def standardize_data(df_in):
    """
    Input:
        data_input: a string data. e.g. %, $,...
    Description:
        Convert the strings in US Census to float.
    Return:
        data_output: a float which convert by string.
    """
    if df_in in ['-','D','F','FN','NA','S','X','Z']:
        df_out = 0
    elif '%' in df_in:
        df_out = float(df_in[0:-2])/100
        
    elif '$' or ',' in df_in:
        df_out = ''
        for char in df_in:
            if (char != '$') and (char !=','):
                df_out += char
        df_out = float(df_out)
    return df_out
